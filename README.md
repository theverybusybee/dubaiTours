## Запуск приложения

## Клонирование репозитория 

```bash
$ git clone git@github.com:theverybusybee/dubaiTours.git
```

#### install dependencies
```bash
$ npm install
```

#### build project
```bash
$ npm run build
```

#### run prod mode
```bash
$ npm run dev
```

## Запуск приложения

Запустите [http://localhost:3000](http://localhost:3000), чтобы отобразить результат запуска.
